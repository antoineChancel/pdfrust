# PdfRust

Pdf parser in pure Rust to extract data.

References : https://opensource.adobe.com/dc-acrobat-sdk-docs/pdfstandards/pdfreference1.7old.pdf

## Usage

```sh
cargo run <pdf_file>.pdf
```
